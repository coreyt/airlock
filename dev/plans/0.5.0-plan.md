# Airlock 0.5.0 — Plan

Release theme: **large-context resilience + the admin/auth control plane.** Two
joined workstreams shipped as one train: (1) stop Airlock's circuit breaker from
amplifying provider 429s into self-inflicted outages and make rate-limit behavior
observable + client-friendly; (2) give operators a control plane (clear/arm
quarantine, reset circuits) plus a capability layer (loopback + JWT) and native
TLS, and let trusted clients downgrade specific guardrails per request.

> Folds the admin/auth/TLS/guardrail-skip work into the **same** release as the
> resilience pack (one train, not a separate admin release). 0.4.0 (the batch
> gateway) is **complete** — see `STATUS-0.4.0.md`; this is the next minor.

Orchestrated with the agent harness — see `dev/agent-harness-runbook.md` and this
directory's `README.md`. Live board: `dev/plans/runs/STATUS-0.5.0.md`.

> **Third workstream — transparency.** A follow-on set of seven packs (mutation &
> serving-backend transparency, UN-19/UN-20) stacks on this branch after the ten
> resilience+admin packs. Plan: `dev/plans/0.5.0-transparency-plan.md`; board:
> `dev/plans/runs/STATUS-0.5.0-transparency.md`; design:

## ◆ HITL gates

- **ADM-http** — admin auth + a new ASGI perimeter; config changes need a proxy
  restart the agent can't perform. After merge with green no-network units, the
  **operator** smoke-tests: loopback clear, JWT clear, 404-when-disabled, and a
  TLS handshake (`AIRLOCK_SSL_*` set).
- **ADM-skip** — capability skip weakens a guardrail; operator confirms the
  downgrade-to-observe + non-skippable-PII posture on real traffic before GA.
- **Release tag** — HITL-only, after all packs land and the smoke-tests pass.

## Cross-cutting constraints (every pack)

- **TDD**: write/extend tests as you go; match existing pytest style; new unit
  tests **no-network** (`uv run python -m pytest -q`).
- **CC-3 / off-by-default**: a config-free deploy must behave exactly as today.
- **Mount order (§3 of the umbrella)**: never reorder the batch gateway; admin
  perimeter installs before it; the error handler is terminal.
- **Process split (CC-12)**: TLS in the parent `proxy.py` builder; everything else
  in the subprocess store injector.
- **Do not run `uv sync`** without re-running `make ensure-spacy`.
- **Config changes need a proxy restart** — hand live verification to the operator.
- **Trust boundary**: a capability skip can never disable provider-protection or
  re-enable fallbacks (CC-10).
