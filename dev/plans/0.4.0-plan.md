### §7 open design decisions

§7 = section 7 of `dev/design-unified-batch-gateway.md` ("Status of the holes +
remaining open items") — six things deliberately left to decide during the build:

1. **Async-scan UX** — failure UX when a scan rejects *after* `/v1/files` already
   returned 200. (→ belongs to item 2, content-scan.)
2. **PII hydration for batch** — support it at all, or terminal-redaction only?
   (The encrypted, TTL'd, ≤48h bulk-PII store is the cost.) (→ deferred;
   terminal-redaction only for 0.4.0.)
3. **Result-file retention ≠ job expiry** — the provider result file can expire
   independently of the 48h job window; don't assume it's fetchable the whole
   time. (**Intersects the gateway build — Pack C.**)
4. **Verify `airlock_batch` placement** — under `litellm_params` it might leak to
   the provider SDK on the sync path; verify, don't assume. (**Intersects the
   gateway build — Pack C.**)
5. **Webhooks** — need a public HTTPS callback + signature verification; polling
   is the MVP, **webhooks deferred**.
6. **`LiteLLMNativeBackend` double-handling** — if the gateway later fronts the
   wired providers, avoid gateway-scans-then-litellm-reprocesses. (→ deferred.)

Resolved in-band this release: **#3 and #4** (Pack C). The rest belong to item 2
(content-scan) or are deferred.

## ◆ HITL gates

- **Pack C live e2e** — config changes need a proxy restart the agent cannot
  perform, and the e2e hits the network (AI Studio). After C merges with green
  no-network unit tests, the **operator** runs the live e2e: upload a tiny JSONL
  → create a batch for `gemini-3.5-flash` via AI Studio → poll to completed →
  download OpenAI-shaped results (provider-native kept verbatim in `.body`).
- **Release tag** — HITL-only, after items 1/3/4 land and the e2e passes.

## Cross-cutting constraints (every pack)

- **TDD**: write/extend tests as you go; match existing pytest style. New unit
  tests must be **no-network**.
- **Lazy-import provider SDKs** inside handlers so the proxy boots without the extra.
- **Do not run `uv sync`** without re-running `make ensure-spacy` (it prunes
  `en_core_web_lg`).
- **Config changes need a proxy restart** to take effect — the agent can't
  restart; hand that to the operator.
- **Trust boundary**: ignore caller-supplied guardrail/metadata disables on the
  batch path.
