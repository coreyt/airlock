     per-line `error` case like aistudio.
   - **Status map** Mistral → OpenAI batch vocab: QUEUED→`validating`,
     RUNNING→`in_progress`, SUCCESS→`completed`, FAILED→`failed`,
     TIMEOUT_EXCEEDED→`expired`, CANCELLATION_REQUESTED→`cancelling`,
     CANCELLED→`cancelled`; unknown → `in_progress` (never crash a poller).
   - **Lazy SDK** (`mistralai`, imported inside the handler; clear "install the
     mistral extra" error if missing). Auth `MISTRAL_API_KEY`.

     "batch", file=…)`); `create(model, file_ref, display_name)` →
     `batch.jobs.create(input_files=[file_ref], model=provider_model or model,
     endpoint="/v1/chat/completions", metadata={"display_name": display_name})`;
     `poll` → `batch.jobs.get`; `fetch` → download the job's OUTPUT file via
     `files.download`, raise `ResultUnavailableError` if missing/expired (§7.3);
     `cancel` → `batch.jobs.cancel`; `list_jobs(display_name)` → `batch.jobs.list()`
