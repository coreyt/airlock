     from `guardrails/extract.py` into `airlock/text_extract.py`. Make
     `guardrails/extract.py` re-export them (`from airlock.text_extract import *` plus
     an explicit `__all__`) so every existing `airlock.guardrails.extract` importer
     keeps working unchanged.
   - Move `resolve_guardrail_decision`, `configure_guardrail_overrides`, and the
     overrides dataclasses from `guardrails/overrides.py` into
     `airlock/guardrail_overrides.py`; make `guardrails/overrides.py` a re-export shim
     (keeps `proxy.py:383` + `keyword_guard.py:86` working untouched).

     import from the neutral modules (`airlock.text_extract` / `airlock.guardrail_overrides`).
