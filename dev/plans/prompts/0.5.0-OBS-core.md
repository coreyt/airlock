## Design Decisions (already resolved — you CANNOT infer these; do not redesign)

This module realizes §3 of the design note verbatim. Key points:

**1. Dataclasses (use `from __future__ import annotations`; `@dataclass(slots=True)`
is fine on 3.10; `Literal`/`Any` from typing):**

```python
MutationOp = Literal["set", "drop", "clamp", "rewrite", "inject", "redact", "suppress"]
MutationStage = Literal["pre_call", "during_call", "post_call"]

@dataclass(slots=True)
class Mutation:
    field: str
    op: MutationOp
    before: Any | None
    after: Any | None
    stage: MutationStage
    source: str
    reason: str | None = None
    count: int | None = None      # redact-only
    category: str | None = None   # redact-only
    def __post_init__(self):
        # CC-T2 invariant — redaction records can NEVER carry the matched value.
        if self.op == "redact" and (self.before is not None or self.after is not None):
            raise ValueError("redact mutations must be value-free; use record_redaction()")

@dataclass(slots=True)
class ServedBackend:
    provider: str | None
    api_base_host: str | None
    region: str | None
    model_id: str | None
    response_cost: float | None
    backend_kind: Literal["native", "gateway", "unknown"]
```

(Match the exact field names/order above — later packs and the JSONL log schema in
§3 of the design depend on them. `reason`/`count`/`category` are keyword defaults.)

**2. CC-T2 header-safety allowlist (module constant):**
```python
HEADER_VALUE_FIELDS = {"model", "reasoning_effort", "fallbacks", "num_retries"}
```
The header serializer surfaces an `after` VALUE **only** for fields in this set;
every other op/field (`inject`/`rewrite`/`redact`, message/system-prompt content)
renders as `field=<op>` (or `field=<op>(<count>)` for redact) — **never** content.

**3. Helpers (exact signatures from §3):**
```python
def record_mutation(metadata: dict, *, field, op, before=None, after=None,
                    stage, source, reason=None) -> None
    # appends a Mutation to metadata.setdefault("airlock_mutations", []) (CC-T1: one ledger)
def record_redaction(metadata: dict, *, field, count, category, stage, source) -> None
    # value-free redact record (op="redact", before/after stay None) — the ONLY way redactions are recorded
def attribute_served_backend(response, *, cost_fallback=None) -> ServedBackend | None
def detect_dropped_params(data: dict, model: str, provider: str) -> list[str]
def mutations_header(ledger, budget_bytes: int = 256) -> str
def served_headers(s: ServedBackend) -> dict[str, str]
```
- `record_mutation`/`record_redaction` mutate `metadata` in place, appending to the
  single list `metadata["airlock_mutations"]`, preserving call order. They store
  `Mutation` instances (later packs serialize). `record_redaction` must route through
  the `Mutation` ctor so the CC-T2 invariant is enforced (no value ever stored).
- **`attribute_served_backend(response, *, cost_fallback=None)`** (§3 + §4.1 — must
  tolerate partial reads):
  - Read `hp = getattr(response, "_hidden_params", {}) or {}`.
  - **Provider (CC-T3, streaming-correct):**
    `provider = hp.get("custom_llm_provider") or getattr(response, "custom_llm_provider", None)`.
    On streams the provider is a wrapper *attribute*, not yet in `_hidden_params`.
  - `api_base_host` = host only, parsed from `hp.get("api_base")` via
    `urllib.parse.urlsplit(...).hostname` (None-safe; None when no api_base).
  - `region` = `hp.get("region_name")`.
  - `model_id` = `hp.get("model_id") or hp.get("litellm_model_name") or hp.get("received_model_id")`.
  - `response_cost` = `hp.get("response_cost")` if present **else** `cost_fallback`
    (None at the header hook on streams; final in the log hook — that's why it's Optional).
  - **`backend_kind`** derived from provider: `"gateway"` if provider in
    `{"bedrock", "azure", "vertex_ai"}`; `"native"` if provider in
    `{"anthropic", "openai", "gemini"}`; else `"unknown"`. (Google AI-Studio-vs-Vertex
    is captured by provider itself — `gemini`=native AI Studio vs `vertex_ai`=gateway —
    plus the recorded `api_base_host`, e.g. `generativelanguage.googleapis.com` for
    AI Studio. Do NOT special-case hostnames for backend_kind beyond this mapping.)
  - **If provider is None** (pre-call block / transport error / unknown at header
    time): still return a `ServedBackend` with `provider=None` and `backend_kind="unknown"`
    (partial), so the log hook can record `attribution: "inferred"`. The caller decides
    to omit the header. Return `None` only when `response` itself is falsy/None.
- **`served_headers(s)`** returns `{}` when `s is None or s.provider is None` (omit,
  never guess); otherwise `{"X-Airlock-Served-By": s.provider}` plus
  `"X-Airlock-Served-Region": s.region` **only when** `s.region` is truthy. (Header
  name is `X-Airlock-Served-Region`.)
- **`mutations_header(ledger, budget_bytes=256)`**: render each Mutation to a compact
  token, join with `;`. Allowlist rule per Decision 2:
  - field in `HEADER_VALUE_FIELDS` AND op in `{"set","clamp","rewrite"}` for a scalar
    `after` → `field=<after>` (e.g. `reasoning_effort=minimal`, `model=claude-sonnet`,
    `num_retries=0`). For `op="suppress"` on an allowlisted field → `field=suppressed`
    (e.g. `fallbacks=suppressed`).
  - op `"redact"` → `field=redacted(<count>)` (count from the record; e.g.
    `messages=redacted(3)`) — never a value.
  - everything else (`inject`, `rewrite`/`drop` on non-allowlisted fields, or any
    non-allowlisted field) → `field=<op>` (e.g. `system=inject`, `temperature=drop`) —
    never content.
  - **Byte-bound:** if the joined string exceeds `budget_bytes` (UTF-8), truncate to
    the most tokens that fit alongside a trailing `…+N more` suffix (N = dropped
    count). The full ledger always survives in the JSONL log (a later pack); the
    header is lossy by design.
- **`detect_dropped_params(data, model, provider)`** (Decision 8): compute
  `supported = set(litellm.get_supported_openai_params(model=model, custom_llm_provider=provider) or [])`
  and return the client-supplied OpenAI params present in `data` but absent from
  `supported`. Restrict the comparison to a fixed set of droppable OpenAI params
  (e.g. iterate the keys of `data` that are recognized OpenAI request params —
  `temperature, top_p, n, stream, stop, max_tokens, presence_penalty,
  frequency_penalty, logit_bias, user, response_format, seed, tools, tool_choice,
  reasoning_effort, logprobs, top_logprobs` — intersect with `data`’s keys), so
  metadata/airlock-internal keys are never flagged. Return a list[str] of dropped
  param names, order stable. This is a pure local lookup — **no network**. If
  `get_supported_openai_params` raises for an unknown model/provider, return `[]`
  (best-effort; never raise out).
