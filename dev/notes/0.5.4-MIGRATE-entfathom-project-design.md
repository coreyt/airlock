# 0.5.4-MIGRATE-entfathom-project — design memo

## Goal
Add two PURE projection functions over the canonical `RequestEvent` and prove,
golden / field-for-field, that they reproduce today's
`AirlockLogger._build_record` and fathom `_fathom_properties` output. Strictly
additive: no sink/builder/registration/config edits. The old builders stay in
place and serve as the equivalence oracle (rewiring is pack 2b).

## `project_enterprise(event) -> dict`
Mirrors `_build_record` (:483-539) field-for-field, sourcing each field from the
event instead of re-deriving:
- base scalars from event fields (`timestamp`, `record_type="request"`, `success`,
  `model`, `user`, `team`, `request_id`, `messages`, `error*`, `failure_category`,
  `airlock_provider`, `start_time`/`end_time`, `duration_ms`);
- `response` = `_serialize(event.response_obj) if event.response_obj else None`
  (reuse enterprise `_serialize` on the SAME raw object the event carries);
- spread `**event.usage`, `**event.mcp_meta`, `**event.guardrail_meta` (builder
  key order :508-510);
- then `mutations`, `served`, `attribution`, `airlock_client` last (builder
  :517-538).
Only accepted divergence vs the live builder: `timestamp` (registered
convergence §3.4) — golden test compares ignoring `timestamp`.

## `project_fathom(event) -> dict`
Mirrors `_fathom_properties` (:121-172) from the event; env-flag gating stays in
the projection. Reuses fathom `_json_text`, `_response_text`, `_env_flag`.
- base props from the event; `error_flag = not event.success`;
  `call_id = event.request_id`; tokens from `event.usage` (`.get(..., 0)`);
  `cost = event.response_cost if event.response_cost is not None else 0` (matches
  `kwargs.get("response_cost", 0)`); `call_type/mcp_tool_name/mcp_server_name`
  from `event.mcp_meta.get(...)` (absent → None → filtered);
- env-gated additions sourced from the event
  (`airlock_client`, `user`/`team`, `error_type`/`error`,
  `messages_json=_json_text(event.messages)`,
  `response_text=_response_text(event.response_obj)`,
  `headers_json=_json_text(event.request_headers)`,
  `mcp_arguments_json=_json_text(event.mcp_arguments)`);
- final None-filter `{k:v for k,v in props.items() if v is not None}`.

## Test strategy (`tests/test_projections_equiv.py`, no network)
Representative request set: plain success, provider failure, pre-call failure,
MCP call, Gemini success, mutations+served, airlock_* guardrail keys. For each:
build the event and call the OLD function from the SAME inputs (fresh deepcopy of
kwargs per call so the in-place Gemini enrich can't cross-contaminate; the raw
response object is shared so `_serialize`/`str(obj)` is identical on both sides),
assert projection == old after deleting `timestamp` from both. Fathom runs over
the 7-flag `AIRLOCK_FATHOM_STORE_*` matrix (all-off, all-on, mixed) via
`monkeypatch.setenv`.
