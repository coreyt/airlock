# 0.5.3-ACL — design memo

Pure extraction: introduce `airlock/litellm_adapter.py` as the single Anti-Corruption
Layer that owns every read of LiteLLM-internal surfaces. Zero behavior change —
byte-parity is the bar.

## Adapter public surface (`airlock/litellm_adapter.py`)

Response-internal accessors (LiteLLM 1.89.0 `_hidden_params` contract lives here):

- `hidden_params(response) -> dict | None`
  `getattr(response, "_hidden_params", None)` — the raw private dict (or None).
- `served_provider(response) -> str | None`
  `hp.get("custom_llm_provider") or getattr(response, "custom_llm_provider", None)`
  (streaming wrapper hardcodes `vertex_ai_beta`; normalization stays in
  `transparency._normalize_served_provider`).
- `response_cost(response, fallback=None) -> float | None`
  `hp.get("response_cost")`, falling back to `fallback` when None.
- `additional_headers(response) -> dict | None`
  read accessor for the nested `additional_headers` dict.
- `merge_additional_headers(response, headers) -> bool`
  write seam: `hp.setdefault("additional_headers", {}).update(headers)` when hp is a dict.

Proxy-app surface:

- `resolve_proxy_app() -> Any`
  `getattr(sys.modules.get("litellm.proxy.proxy_server"), "app", None)`.
- `install_asgi_middleware(app, middleware_cls, *args, **kwargs) -> None`
  encodes the `middleware_stack is None` (use `add_middleware`) vs built-stack
  (wrap it) branch ONCE. Install ORDER stays in callers (Q2).

## Intentionally excluded (Q1/Q3/Q4 — stay out of ACL)

- `callbacks/enterprise_logger.py:523`, `callbacks/fathom_logger.py:132`,
  `fast/monitor.py:163,173,186` bare `kwargs.get("response_cost")` — documented
  CustomLogger kwargs surface, not `_hidden_params` (Q1).
- `batch/middleware.py:48-65` `custom_llm_provider` query-param parsing — client
  request surface (Q3).
- `enterprise_logger.py:_patch_lowest_cost_none_guard` — router-strategy monkey-patch (Q4).
- `providers/enhanced_passthrough.py:169` — mutates request params, not an internal read.
