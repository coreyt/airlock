# 0.5.0-OBS-core ‚Äî implementation memo

**Pack:** 0.5.0-OBS-core. **Branch:** obs-core (base 96744f2).

## Goal

Ship a new, pure module `airlock/transparency.py` ‚Äî the mutation-ledger +
served-backend-attribution dataclasses/helpers, header serializers, and the
`transparency.*` config loader ‚Äî plus a documented `transparency:` block in
`config.yaml`. **Zero call-sites change.** Behavior is byte-for-byte identical;
later packs (OBS-served, OBS-ledger, ‚Ä¶) import this module and wire it in.

Realizes ¬ß3 of `dev/notes/design-mutation-and-provider-transparency.md` verbatim.

## Contracts (authoritative ‚Äî later packs depend on field names/order/signatures)

### Dataclasses
- `Mutation(field, op, before, after, stage, source, reason=None, count=None,
  category=None)` ‚Äî `@dataclass(slots=True)`. `__post_init__` enforces CC-T2:
  `op == "redact"` with non-None `before`/`after` raises `ValueError`.
- `ServedBackend(provider, api_base_host, region, model_id, response_cost,
  backend_kind)`.
- `MutationOp = Literal["set","drop","clamp","rewrite","inject","redact","suppress"]`
- `MutationStage = Literal["pre_call","during_call","post_call"]`
- `HEADER_VALUE_FIELDS = {"model","reasoning_effort","fallbacks","num_retries"}`

### Helpers
- `record_mutation(metadata, *, field, op, before=None, after=None, stage, source,
  reason=None)` ‚Äî append a `Mutation` to `metadata.setdefault("airlock_mutations", [])`.
- `record_redaction(metadata, *, field, count, category, stage, source)` ‚Äî routes
  through the `Mutation` ctor (op="redact", value-free) ‚Äî the only redaction path.
- `attribute_served_backend(response, *, cost_fallback=None) -> ServedBackend | None`
  ‚Äî None only when `response` falsy; provider from `_hidden_params["custom_llm_provider"]`
  OR wrapper attr; backend_kind map gateway={bedrock,azure,vertex_ai},
  native={anthropic,openai,gemini}, else unknown; cost falls back to `cost_fallback`.
- `detect_dropped_params(data, model, provider) -> list[str]` ‚Äî droppable OpenAI
  params in `data` absent from `litellm.get_supported_openai_params(...)`. Never
  raises out (returns `[]` on lookup failure). No network.
- `mutations_header(ledger, budget_bytes=256) -> str` ‚Äî allowlist-aware, byte-bounded
  with `‚Ä¶+N more`. Values only for HEADER_VALUE_FIELDS+{set,clamp,rewrite}; suppress ‚Üí
  `field=suppressed`; redact ‚Üí `field=redacted(<count>)`; else `field=<op>`.
- `served_headers(s) -> dict[str,str]` ‚Äî `{}` when `s` None or `s.provider` None;
  else `X-Airlock-Served-By` + `X-Airlock-Served-Region` (only when region truthy).

### Config
- `TransparencyConfig` frozen dataclass, one field per `_DEFAULTS` key.
- `load_transparency_config(config)` reads top-level `config["transparency"]`,
  applies defaults, validates (`mutation_headers ‚àà {off,compact,full}`, budget a
  positive int, booleans coerced). Unknown/missing ‚Üí defaults (CC-T7).
- `configure_transparency(config)` / `get_transparency_config()` ‚Äî module-cached
  accessor; returns safe defaults before `configure_transparency` is ever called.
  **Not** called from proxy startup here (OBS-served owns wiring).
